# 🌰 Project Acorn

> **Production Started!** 🚀

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Desktop Framework:** Tauri
- **Package Manager:** Bun

## 🚀 Quick Start

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```
